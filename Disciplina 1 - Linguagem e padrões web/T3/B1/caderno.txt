Podemos utilizar o <script> = usado para o navegador interpretar o que será entendido como "Javascript"

Pegar um elemento iD, HTML pelo JS:

	document.getElementById("paragrafo").innerText = "Hello World";
	
Conectar um script JS no HTML:

<script src="script.js"></script>
